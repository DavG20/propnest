# Propnest

A professional real estate application.

## Structure

- `frontend/`: Next.js web application
- `backend/`: Go REST API

## Getting Started

### Prerequisites
- Go 1.21+
- Node.js 18+
- Docker & Docker Compose

### Running locally
Use the provided Makefile to start the services:

```bash
# Start everything
make up

# Stop everything
make down
```
